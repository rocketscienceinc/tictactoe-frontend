# tictactoe-frontend

## Requirements
- Node.js (v20.17.0)
- npm (v10.8.2)
  
## Development
1. Clone the repository:
```bash
git clone https://github.com/rocketscienceinc/tictactoe-frontend.git
```
2. Navigate to the project directory:
```bash
cd tictactoe-frontend
```
3. Install dependencies:
```bash
npm install
```
4. Start the project:
```bash
npm start
```
or
```bash
npm run dev
```
